# Реализация QAM-4, 16, 64 модулятора и демодулятора с добавлением АБГШ

В репозитории 3 класса:
- QamMod - модулятор
- AWGN - добавление АБГШ к созвездию
- QamDemod - демодулятор

### Файл QAM.cpp

Это main файл, в котором реализован последовательный вызов объектов классов для случайной последовательности бит для разных значений дисперсий шума. 

Также в этом файле с помощью **gnuplot** построен график зависимости вероятности ошибки на бит от дисперсии шума.
